# smallestSvelteTodo
The final applet from my [A functional todo applet in Svelte 5, using less than 15 lines of code?](https://madr.blog/posts/a-functional-todo-applet-in-svelte5-using-less-than-15-lines-of-code/) blog post.

## Usage
```sh
pnpm i

pnpm run dev

pnpm run build
```
